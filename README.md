# NLP_Projects

Sentiment Analysis Using NLP

Developed a Natural Language Processing (NLP) model to classify text reviews as positive, negative, or neutral. Performed text preprocessing, tokenization, stopword removal, feature extraction using TF-IDF, and trained machine learning models for sentiment classification. Built an interactive application for real-time sentiment prediction.
Tech Stack: Python, NLTK, Pandas, NumPy, Scikit-learn, TF-IDF, Streamlit.
